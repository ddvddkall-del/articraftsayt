ARTICRAFT UMUMIY SERVER QOIDALARI

1. Asosiy qoidalar

1.1. Serverdagi barcha o‘yinchilar teng huquqli.

1.2. Hacked clientlardan foydalanish taqiqlanadi (Xray, KillAura, Reach, Fly va boshqa chitlar) Harakatni o‘zgartiruvchi modlar taqiqlanadi (Speed, Sprint modlari va boshqalar) Inventar bilan bog‘liq modlar taqiqlanadi (Auto sort, inventory tweak va hokazo) Radar modlari taqiqlanadi Freecam ishlatish taqiqlanadi Auto place / Easy place modlari taqiqlanadi Makros va skriptlardan foydalanish taqiqlanadi Server baglaridan foydalanish yoki suiiste’mol qilish taqiqlanadi Ko'rinmas odamning xpsni va nikini ko'rsatadigon modlar taqiqlanadi Dupe (narsani ko‘paytirish) usullaridan foydalanish taqiqlanadi Haqiqiy pulga savdo qilish taqiqlanadi Odam taklif qilib mukofot berish taqiqlanadi Boshqa serverlar bilan savdo qilish taqiqlanadi Server seedini topish yoki undan foydalanish taqiqlanadi Crafting jarayonini o‘zgartiruvchi modlar taqiqlanadi O‘zingizni server ma'muriyati (admin/moderator) deb ko‘rsatish taqiqlanadi Tashqi kazino (pul tikish o‘yinlari) taqiqlanadi shularning xammasi man etiladi. → Jazo: 15—kun ban

1.3. Serverda lag, dupe yoki bug ishlatish. 30—kun ban.

1.4. “Qoida bilmaslik” — javobgarlikdan ozod qilmaydi.

1.5. O‘yin paytida /spawn, /home, yoki /tpa orqali PvPdan qochish — man etilmagan server qochishga yoʻlb qoymaydi

1.6. Boshqa o‘yinchilarga ruhiy yoki moddiy zarar yetkazish. 1—soat ban

1.7. Serverda tartib, adolat va hurmat saqlanishi shart.

1.8. Har qanday server fayliga, ma’lumotiga yoki sistemaga zarar yetkazish — cheksiz ban.

1.9. Katta moder va admin qarori yakuniy hisoblanadi.

1.10. Moderator sizga ban bersa sababni oldida telegram useri bo'ladi.

2. 💬 Chat qoidalari

2.1. Chatda so‘kinish, haqorat, 3—soat mute yoki.

2.2. Spam, flood, bilan yozish. 2—soat mute.

2.3.Boshqa odamlarning ma'lumotlarini tarqatish. 5—soat mute.

2.4. Reklama, link, IP yoki boshqa server nomini yozish. 30—kun ban.

2.5. Moderator va adminni haqoratlash. 3—kun ban yoki.

2.6. Oila Aʼzolaridan Soʻkinish. 1—kun ban.

2.7. Chatda Chiterlik tan olish. 15—kun ban.

2.8. Server ma'muryatiga yolg'on gapirish. 3—kun ban

2.9. Chatda faqat madaniyatli muloqotga ruxsat beriladi.

3. ⚔️ O‘yin qoidalari

3.1. Jamoa 3 kishidan ko'p bo'lishi taqiqlanadi. PvPda faqat 3 kishilik Jamoa Tarkibida Ruxsat etiladi. 3 kishidan ko‘p bo‘lib Harakat qilish, PvP qilish, Baza raid qilish (qidirish), warplarga borish (boss, pvp va hokazo) va airdroplarga borish, va shunga oxshagan holatlar taqiqlanadi. 3—Kun Ban..

3.2. Bug, lag, dupe ishlatish. 30kun—ban.

3.3 Teleport trap (o‘yinchini chaqirib o‘ldirish) —taqiqlanmagan.

3.4. Serverni lagga olib keluvchi mashina yoki qurilma yasash —♾️ban.

3.5. Eventlarda tartibni buzish — ♾️ban.

3.6. Agarda Uyinchilar Aldab savdo qilsa — Server javobgar emas.

4. 👤 Akkaunt qoidalari

4.1. Oʻyinchi telegram linkda 3tadan ko'p akkaunt ochishi taqiqlanadi.

4.2. Akkauntni sotish, berish yoki almashish — cheksiz ban.

4.3. Parolni hech kimga bermang, hatto admin ham so‘ramaydi.

4.4. Akkauntdagi donat, narsalar, world yo‘qolsa — o‘yinchi javobgar.

4.5. Nickda haqorat, so‘kinish yoki reklama bo‘lmasin — cheksiz ban.

5. 🔍 Tekshiruv (Proverka) qoidalari

5.1. Moderator sizni aftamitichski proverkaga chaqiradi.

5.2. Tekshiruvdan qochish (chiqib ketish, o‘chirish) — 15kun ban aftamatichiski.

5.3. Fayl, folder, client yashirish yoki o‘chirish — 15kun ban.

5.4. RustDesk / AnyDesk orqali proverkaga ruxsat berish kerak.

5.5. Tekshiruvda dalil topilsa — 15kun ban.

5.6. Proverkadan qochish.15—kun ban.

5.7. Moderatorga bosim yoki tahdid 1—kun ban.

5.8. Tekshiruvda toza deb topilgan o‘yinchi tiklanadi.

5.9. Har bir tekshiruv faqat moderator ruxsati bilan o‘tkaziladi.

5.10. Moderatorga pora berish — 15kun ban

6.💎 Donat qoidalari

6.1. Donat (ixtiyoriy to‘lov) — serverni qo‘llab-quvvatlash maqsadida foydalanuvchilar tomonidan beriladigan pul mablag‘idir.

6.2. Donat qabul qilish va donat qiluvchi hamma qoidalar va shartlarga rioya qilishi majburiy. Donatorlar boshqa o‘yinchilardan alohida immunitetga ega emas.

6.3. Donat to‘lovlari qaytarilmaydi — barcha donatsiyalar so‘nggi va qaytarilmas hisoblanadi.

6.4. Donat qilish paytida o‘yinchi chek/kvittansiya/transaction ID ko‘rsatishi majburiydir. Chekni rasmiy support-kanalga yoki ma’muriyat belgilagan joyga yuborish talab qilinadi.

6.5. O'yinchi chekni saqlashi va ma’muriyat talab qilgan taqdirda uni taqdim etishi shart. Chek taqdim etilmasa — donat qabul qilinmasligi yoki imtiyozlar berilmasligi mumkin.

6.6. Ma’muriyat hech qachon parol yoki shaxsiy ma’lumot so‘ramaydi; agar shunday so‘rov bo‘lsa — bu firibgarlik hisoblanadi va darhol xabar qilinishi lozim.

6.7. Donat orqali olingan imtiyozlar (Rank, kosmetika, yoki tokenlar) faqat akkauntga bog‘langan bo‘ladi va o‘tkazilmaydi, sotilmaydi yoki berilmaydi.

6.8. Donat serverda qoida buzilishiga yo‘l qo‘yishda himoya yoki ruxsat bermaydi — qayd etilgan har qanday qoidabuzarlik uchun donator ham normal jazo choralariga tortiladi.

6.9. Ma’muriyat istalgan vaqtda donat imtiyozini shartlarga zid ishlatilsa yoki qoidabuzarlik aniqlansa qo‘lga olishi va bekor qilishi mumkin.

6.10. Donatni sotish, berish yoki o‘tkazish qat’iyan taqiqlanadi. Donat orqali olgan imtiyoz yoki pullarni uchinchi shaxsga o‘tkazish aniqlansa — Jazo: Umrbod ban.

6.11. Soxta to‘lov yoki soxta chek (falsifikatsiya qilingan kvittansiya, yolg‘on transaction ID va h.k.) ishlatish — katta jinoyat hisoblanadi. Jazo: Umrbod ban (permanent ban).

6.12. Agar foydalanuvchi to‘lov qilganini tasdiqlash uchun chekni taqdim etsa-da, keyinchalik to‘lovni bank/ta’minotchi orqali chargeback qilib qaytarib oladi (orqaga tortadi), bu holat ham soxta/aldash deb hisoblanadi va jiddiy jazo bilan yakunlanadi (umrbod ban va imtiyozlarning bekor qilinishi).

ARTICRAFT UMUMIY SERVER QOIDALARI

1. Asosiy qoidalar

1.1. Serverdagi barcha o‘yinchilar teng huquqli.

1.2. Har qanday chit yoki x-ray qat’iyan man etiladi. Jazo: 30 kun ban.

1.3. Serverda lag, dupe yoki bug ishlatish — cheksiz ban.

1.4. Moderator yoki admin so‘zini inkor etish — 3 kun ban.

1.5. “Qoida bilmaslik” javobgarlikdan ozod qilmaydi.

1.6. PvP paytida /spawn, /home, /tpa orqali qochish — man etilmagan.

1.7. Boshqa o‘yinchilarga ruhiy yoki moddiy zarar yetkazish — 1 kun ban.

1.8. Serverda tartib, adolat va hurmat saqlanishi shart.

1.9. Server fayli yoki sistemaga zarar yetkazish — cheksiz ban.

1.10. Admin qarori — yakuniy.

2. Chat qoidalari

2.1. So‘kinish, haqorat, diniy yoki milliy kamsitish — 5 soat mute yoki ban.

2.2. Spam, flood, CAPS bilan yozish — 2 soat mute.

2.3. Reklama, link, IP yoki boshqa server nomini yozish — cheksiz ban.

2.4. Moderator yoki adminni haqoratlash — 10 kun ban yoki mute.

2.5. Shaxsiy ma’lumot so‘rash yoki tarqatish — man etilgan.

2.6. Oila a’zolaridan so‘kinish — 3 kun ban.

2.7. Chatda faqat madaniyatli muloqot ruxsat etiladi.

3. O‘yin qoidalari

3.1. Jamoa 3 kishidan ko‘p bo‘lib PvP qilsa — 5 kun ban.

3.2. Bug, lag, dupe orqali foyda olish — cheksiz ban.

3.3. Teleport trap (chaqirib o‘ldirish) — man etilmagan.

3.4. Lag keltiruvchi qurilma yasash — ban.

3.5. Eventlarda tartibni buzish — ban.

3.6. Aldab savdo qilish — server javobgar emas.

4. Akkaunt qoidalari

4.1. O‘yinchi cheksiz akkaunt ocholadi.

4.2. Akkauntni sotish, berish yoki almashish — cheksiz ban.

4.3. Parolni hech kimga bermang, hatto admin ham so‘ramaydi.

4.4. Narsalar yo‘qolsa — o‘yinchi javobgar.

4.5. Nickda haqorat yoki reklama — cheksiz ban.

4.6. O‘g‘irlangan akkauntlar bloklanadi.

4.7. Himoya uchun /register, /login va Articraft botdan foydalaning.

5. Tekshiruv (Proverka) qoidalari

5.1. Moderator sizni avtomatik proverkaga chaqiradi.

5.2. Tekshiruvdan qochish — 30 kun ban.

5.3. Fayl yoki client yashirish — 35 kun ban.

5.4. RustDesk / AnyDesk orqali ruxsat berish shart.

5.5. Dalil topilsa — 30 kun ban.

5.6. Proverkadan qochgan — “cheater” ro‘yxatiga kiritiladi.

5.7. Moderatorga bosim yoki tahdid — cheksiz ban.

5.8. Toza o‘yinchi — tiklanadi.

5.9. Tekshiruv faqat moderator ruxsati bilan o‘tkaziladi.

6. Donat qoidalari

6.1. Donat to‘lovlari qaytarilmaydi.

6.2. Donatni sotish, berish yoki o‘tkazish — ban.

6.3. Donater ham barcha qoidalarga bo‘ysunadi.

Xulosa

Serverda tartib, hurmat va adolat — hammaning burchi. Articraft — bu o‘yin emas, bu jamoa.
